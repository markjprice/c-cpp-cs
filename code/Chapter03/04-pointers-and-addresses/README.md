# 04 - Pointers and addresses

This project demonstrates the address-of operator `&`, pointer variables, and dereferencing with `*`.

## GCC

```bash
gcc -std=c17 -Wall -Wextra -pedantic main.c -o pointers
./pointers
```

## Clang

```bash
clang -std=c17 -Wall -Wextra -pedantic main.c -o pointers
./pointers
```

## MSVC

```cmd
cl /std:c17 /W4 main.c /Fe:pointers.exe
pointers.exe
```
