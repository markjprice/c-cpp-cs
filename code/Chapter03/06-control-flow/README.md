# 06 - Control flow

This project demonstrates `if`, `else if`, `else`, `switch`, `for`, `while`, `do-while`, `break`, `continue`, labels, and `goto`.

## GCC

```bash
gcc -std=c17 -Wall -Wextra -pedantic main.c -o control_flow
./control_flow
```

## Clang

```bash
clang -std=c17 -Wall -Wextra -pedantic main.c -o control_flow
./control_flow
```

## MSVC

```cmd
cl /std:c17 /W4 main.c /Fe:control_flow.exe
control_flow.exe
```
