# 02 - Data types and variables

This project demonstrates primitive types, initialization, strings, `strncpy`, `struct`, `typedef`, constants, and unsigned integers.

## GCC

```bash
gcc -std=c17 -Wall -Wextra -pedantic main.c -o data_types
./data_types
```

## Clang

```bash
clang -std=c17 -Wall -Wextra -pedantic main.c -o data_types
./data_types
```

## MSVC

```cmd
cl /std:c17 /W4 main.c /Fe:data_types.exe
data_types.exe
```
