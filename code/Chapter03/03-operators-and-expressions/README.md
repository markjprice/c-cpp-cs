# 03 - Operators and expressions

This project demonstrates arithmetic, relational, logical, bitwise, assignment, increment, decrement, ternary, and precedence examples.

## GCC

```bash
gcc -std=c17 -Wall -Wextra -pedantic main.c -o operators
./operators
```

## Clang

```bash
clang -std=c17 -Wall -Wextra -pedantic main.c -o operators
./operators
```

## MSVC

```cmd
cl /std:c17 /W4 main.c /Fe:operators.exe
operators.exe
```
