# 07 - Functions and times table

This project demonstrates functions with no parameters, one parameter, multiple parameters, return values, and a reusable times table function.

## GCC

```bash
gcc -std=c17 -Wall -Wextra -pedantic main.c -o functions
./functions
```

## Clang

```bash
clang -std=c17 -Wall -Wextra -pedantic main.c -o functions
./functions
```

## MSVC

```cmd
cl /std:c17 /W4 main.c /Fe:functions.exe
functions.exe
```
