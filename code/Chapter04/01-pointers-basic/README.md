# 01-pointers-basic

Demonstrates the difference between a normal variable, its address, a pointer, and dereferencing a pointer.

## Build with GCC

```bash
gcc -std=c17 -Wall -Wextra -pedantic main.c -o pointers_basic
```

## Build with Clang

```bash
clang -std=c17 -Wall -Wextra -pedantic main.c -o pointers_basic
```

## Build with MSVC

```cmd
cl /std:c17 /W4 main.c /Fe:pointers_basic.exe
```

## Run

Linux, macOS, or MSYS2:

```bash
./pointers_basic
```

Windows PowerShell:

```powershell
.\pointers_basic.exe
```
