# 04-struct-arrow-operator

Shows when to use the dot operator (`.`) and when to use the arrow operator (`->`) with structs and pointers to structs.

## Build with GCC

```bash
gcc -std=c17 -Wall -Wextra -pedantic main.c -o struct_arrow_operator
```

## Build with Clang

```bash
clang -std=c17 -Wall -Wextra -pedantic main.c -o struct_arrow_operator
```

## Build with MSVC

```cmd
cl /std:c17 /W4 main.c /Fe:struct_arrow_operator.exe
```

## Run

Linux, macOS, or MSYS2:

```bash
./struct_arrow_operator
```

Windows PowerShell:

```powershell
.\struct_arrow_operator.exe
```
