# Traditional Program class and Main method

This project is a complete runnable version of a code example from Chapter 9, **C# Fundamentals**.

## Build

```bash
dotnet build
```

## Run

```bash
dotnet run
```

## Notes

The project targets .NET 11:

```xml
<TargetFramework>net11.0</TargetFramework>
```

If you are using a different installed SDK while experimenting, change the target framework in the `.csproj` file to the version installed on your computer, for example `net10.0`.
