# 03 - Duplicated AdminUser

Shows the duplication that appears when a second type is similar to an existing type.

## Build with GCC

```bash
gcc -std=c17 -Wall -Wextra -pedantic main.c -o app
```

## Run

```bash
./app
```

## Build with Clang

```bash
clang -std=c17 -Wall -Wextra -pedantic main.c -o app
```

## Build with MSVC

```cmd
cl /std:c17 /W4 main.c /Fe:app.exe
```
