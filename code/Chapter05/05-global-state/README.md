# 05 - Global state

Shows how global variables can be modified from anywhere, making program behavior harder to reason about.

## Build with GCC

```bash
gcc -std=c17 -Wall -Wextra -pedantic main.c -o app
```

## Run

```bash
./app
```

## Build with Clang

```bash
clang -std=c17 -Wall -Wextra -pedantic main.c -o app
```

## Build with MSVC

```cmd
cl /std:c17 /W4 main.c /Fe:app.exe
```
