# 07 - No function overloading

Shows type-specific function names as a workaround for C's lack of function overloading.

## Build with GCC

```bash
gcc -std=c17 -Wall -Wextra -pedantic main.c -o app
```

## Run

```bash
./app
```

## Build with Clang

```bash
clang -std=c17 -Wall -Wextra -pedantic main.c -o app
```

## Build with MSVC

```cmd
cl /std:c17 /W4 main.c /Fe:app.exe
```
