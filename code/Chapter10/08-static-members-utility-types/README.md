# Static members and utility types

Shows static fields, static methods, static constructors, static classes, utility functions, shared state, and the difference between instance and class-level behavior.

## Build and run

From this project folder:

```bash
dotnet build
dotnet run
```

Or from the `ch10-csharp-oop` folder:

```bash
dotnet run --project 08-static-members-utility-types/08-static-members-utility-types.csproj
```

This project targets .NET 11. If you are using a different SDK, update the `TargetFramework` value in the `.csproj` file, for example to `net10.0`.
