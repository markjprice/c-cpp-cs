# Classes and objects

Demonstrates class definitions, object creation, instance data, instance methods, the `this` keyword, required members, and reference behavior.

## Build and run

From this project folder:

```bash
dotnet build
dotnet run
```

Or from the `ch10-csharp-oop` folder:

```bash
dotnet run --project 01-classes-and-objects/01-classes-and-objects.csproj
```

This project targets .NET 11. If you are using a different SDK, update the `TargetFramework` value in the `.csproj` file, for example to `net10.0`.
