# Interfaces and abstraction

Shows abstract classes, abstract methods, interfaces, multiple interface implementation, interface polymorphism, default interface methods, and dependency-injection-style design.

## Build and run

From this project folder:

```bash
dotnet build
dotnet run
```

Or from the `ch10-csharp-oop` folder:

```bash
dotnet run --project 06-interfaces-abstraction/06-interfaces-abstraction.csproj
```

This project targets .NET 11. If you are using a different SDK, update the `TargetFramework` value in the `.csproj` file, for example to `net10.0`.
