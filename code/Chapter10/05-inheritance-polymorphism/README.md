# Inheritance and polymorphism

Demonstrates base and derived classes, `protected`, constructor order, `base(...)`, `virtual`, `override`, runtime polymorphism, polymorphic collections, and sealed classes.

## Build and run

From this project folder:

```bash
dotnet build
dotnet run
```

Or from the `ch10-csharp-oop` folder:

```bash
dotnet run --project 05-inheritance-polymorphism/05-inheritance-polymorphism.csproj
```

This project targets .NET 11. If you are using a different SDK, update the `TargetFramework` value in the `.csproj` file, for example to `net10.0`.
