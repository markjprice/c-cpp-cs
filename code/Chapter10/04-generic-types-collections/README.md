# Generic types and collections

Shows `List<T>`, `Dictionary<TKey,TValue>`, generic methods, multiple type parameters, generic constraints, and collection selection.

## Build and run

From this project folder:

```bash
dotnet build
dotnet run
```

Or from the `ch10-csharp-oop` folder:

```bash
dotnet run --project 04-generic-types-collections/04-generic-types-collections.csproj
```

This project targets .NET 11. If you are using a different SDK, update the `TargetFramework` value in the `.csproj` file, for example to `net10.0`.
