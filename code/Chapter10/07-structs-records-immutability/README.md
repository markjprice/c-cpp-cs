# Structs, records, immutability, and equality

Demonstrates structs, readonly structs, records, value equality, reference equality, and enums.

## Build and run

From this project folder:

```bash
dotnet build
dotnet run
```

Or from the `ch10-csharp-oop` folder:

```bash
dotnet run --project 07-structs-records-immutability/07-structs-records-immutability.csproj
```

This project targets .NET 11. If you are using a different SDK, update the `TargetFramework` value in the `.csproj` file, for example to `net10.0`.
