> **Prompt**: Please show how malloc returns a pointer and how it is used step by step.

When you call `malloc` (Memory Allocation), you are asking the Operating System to carve out a specific number of bytes from the **Heap**. Because `malloc` doesn't know what kind of data you plan to store there (integers, characters, or custom structs), it returns a **void pointer** (`void*`), which is essentially C's way of saying: "Here is a memory address, but you decide what it represents."

---

## Step 1: The Request
You tell `malloc` exactly how much space you need in bytes. To keep your code portable, you should always use the `sizeof` operator.

```c
int *ptr;
ptr = (int*)malloc(sizeof(int)); 
```
* `sizeof(int)` calculates the bytes needed (usually 4).
* `malloc` looks at the Heap for a 4-byte gap.
* `(int*)` is a **cast** that tells the compiler to treat that address as an integer location.

---

## Step 2: The Return (Address Hand-off)
If the OS finds the space, `malloc` returns the **starting memory address** of that block. If it fails (out of memory), it returns `NULL`.



| Variable | Location | Value (Content) |
| :--- | :--- | :--- |
| **`ptr`** | Stack | `0x0050` (The address `malloc` found) |
| **Heap Memory** | Heap (at `0x0050`) | `[Garbage Data]` |

---

## Step 3: Usage (Dereferencing)
The pointer `ptr` now "owns" that spot on the heap. To put data into that spot, you must **dereference** the pointer using the `*` operator.

```c
if (ptr != NULL) {
    *ptr = 100; // Go to address 0x0050 and put 100 there
}
```

---

## Step 4: The Clean-up (`free`)
Unlike variables on the stack, heap memory is never automatically deleted. If you lose the pointer (by overwriting it or letting it go out of scope) without calling `free`, you have a **Memory Leak**.

```c
free(ptr);   // Tells the OS address 0x0050 is now vacant
ptr = NULL;  // Best practice: stop ptr from pointing to "dead" memory
```

## Summary of the "Handshake"
1.  **You:** "I need 4 bytes." (`malloc(4)`)
2.  **OS:** "Found them at address `0x0050`." (Returns pointer)
3.  **You:** "Write `100` to that address." (`*ptr = 100`)
4.  **You:** "I'm done with address `0x0050`." (`free(ptr)`)
