> **Prompt**: Please generate a small C program that uses every type of operator and explain each line.

This program is designed to be a "Swiss Army Knife" of C operators. It covers arithmetic, relational, logical, bitwise, assignment, and miscellaneous operators (like the ternary and `sizeof`) in a single execution flow.

### The Complete "Operator Gauntlet"

```c
#include <stdio.h>

int main() {
    // 1. Arithmetic & Assignment
    int a = 10, b = 3;
    int sum = a + b;        // Addition
    int diff = a - b;       // Subtraction
    int prod = a * b;       // Multiplication
    int quot = a / b;       // Division (integer truncation)
    int rem = a % b;        // Modulo (remainder)

    // 2. Increment/Decrement & Compound Assignment
    a++;                    // Post-increment (a becomes 11)
    b += 2;                 // Compound addition (b = b + 2, so b becomes 5)

    // 3. Relational & Logical (Short-Circuiting)
    // Checks if (11 > 5) AND (5 is not 0)
    int is_valid = (a > b) && (b != 0); 

    // 4. Bitwise Operators
    // a is 11 (1011 in binary), b is 5 (0101 in binary)
    int bit_and = a & b;    // Bitwise AND (0001 = 1)
    int bit_xor = a ^ b;    // Bitwise XOR (1110 = 14)
    int shift = b << 1;     // Left shift (0101 becomes 1010, which is 10)

    // 5. Ternary (Conditional) Operator
    // If a > b, max = a; else max = b
    int max = (a > b) ? a : b;

    // 6. Miscellaneous: sizeof and Address-of
    int size = sizeof(a);   // Returns size of int in bytes (usually 4)
    int *ptr = &a;          // Address-of operator (gets memory location of a)

    // Outputting results
    printf("Arithmetic: %d, %d, %d, %d, %d\n", sum, diff, prod, quot, rem);
    printf("Logical/Relational: %d, Max: %d\n", is_valid, max);
    printf("Bitwise: AND:%d, XOR:%d, Shift:%d\n", bit_and, bit_xor, shift);
    printf("Misc: Size:%d, Pointer Value: %p\n", size, (void*)ptr);

    return 0;
}
```

---

### Detailed Line-by-Line Breakdown

#### 1. Arithmetic Group (`+`, `-`, `*`, `/`, `%`)
* `a / b`: Since both are integers, C performs **integer division**. $10 / 3$ results in $3$, not $3.33$.
* `a % b`: The modulo operator returns the remainder. $10$ divided by $3$ leaves a remainder of $1$.

#### 2. Assignment Group (`=`, `+=`, `++`)
* `b += 2`: This is shorthand. It fetches the value of `b`, adds `2`, and stores it back in `b`.
* `a++`: This increments `a` by `1`. In this specific line, it's "postfix," meaning if it were used in a larger expression, the value would be used *before* the increment happens.



#### 3. Relational & Logical Group (`>`, `&&`, `!=`)
* `(a > b)`: Returns `1` if true, `0` if false.
* `&&`: The logical AND. Because of **short-circuiting**, if `(a > b)` were false, the computer wouldn't even bother checking if `(b != 0)`.

#### 4. Bitwise Group (`&`, `^`, `<<`)
These operate on the individual bits of the numbers.
* `&` (AND): Only sets a bit to `1` if both numbers have a `1` in that position.
* `<<` (Left Shift): Moves all bits to the left. Shifting left by `1` is mathematically equivalent to multiplying the number by $2$.



#### 5. The Ternary Operator (`? :`)
This is the only **trinary** operator in C (it takes three operands). It’s a compact way to write an `if-else` statement. 
* `Condition ? ValueIfTrue : ValueIfFalse`

#### 6. Pointer & Size Operators (`sizeof`, `&`)
* `sizeof`: This is technically an operator, not a function. It's evaluated at **compile-time**.
* `&`: This "points" to the physical address in your RAM where the variable `a` is stored.
